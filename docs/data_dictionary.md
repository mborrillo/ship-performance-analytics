# Data Dictionary

Descripción de los campos utilizados en el proyecto.

| Campo                   | Tipo     | Descripción                                             | Ejemplo    |
|-------------------------|----------|---------------------------------------------------------|-----------:|
| Revenue_per_Voyage_USD  | integer  | Ingresos generados por el viaje en USD                  | 125000     |
| Operational_Cost_USD    | integer  | Costos operativos totales asociados al viaje en USD     | 83000      |
| Efficiency_nm_per_kWh   | float    | Millas náuticas recorridas por cada kWh consumido       | 2.3        |
| Ship_ID                 | texto    | Identificador único del barco                           | SHIP_01    |
| Voyage_Date             | fecha    | Fecha de salida o referencia del viaje                  | 2023-07-15 |
| Route                   | texto    | Ruta o región del viaje                                 | ATL_NORTH  |
| Weather_Condition       | texto    | Condiciones climáticas durante el viaje                 | Stormy     |
